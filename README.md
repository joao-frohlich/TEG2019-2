# TEG2019-2
Repositorio para armazenar os codigos utilizados na materia de TEG

**Visualizador de algoritmos:** [visualgo](visualgo.net)

## Algoritmos implementados

### * Ordenação Topológica:
  * Algoritmo de Kanh

### * Encontrar caminho euleriano:
  * Algoritmo de Fleury

### * Percorrer Árvores:
  * DFS
  * BFS

### * Caminho mínimo:
  * Dijkstra
